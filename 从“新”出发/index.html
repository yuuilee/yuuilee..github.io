从“新”出发

之前我一直从事行政、前台的文职类工作，天天浑浑噩噩，过着得过且过的日子，出来社会这么久自己都不了解自己想要什么，更加不知自己的目标是什么，想法也一直摇摆不定，更不要说对未来的规划了。虽然岗位问题我是有考虑过，对于职业生涯很头疼也很焦虑，并且也尝试业余学习其他来改变现在状态。但还是没有成功，也没有找到突破口。直到 8 月底，物联网部门在招项目助理(协助部门大佬部门管理，项目协调，软件测试等工作)，这个岗位后期发展比较灵活，可以往产品 UI 测试多方面发展。我自己一直想要往更好的方向发展，也觉得互联网发展前景也比较好，刚好现在有一个这么好的机会也可以提升自己，不想错过、也不想在继续过着浑浑噩噩的日子，所以向该部门的大佬申请加入这个团队。

接下来是最重要的事情–学习来提升自己。学习作为一种永无止境的人生追求，一刻也不能放松，前进的动力要从学习中获得，尤其是知识经济时代， 边学习，边工作。用工作带学习，以学习促进工作，在学习中有所发现，有所创造，不断升华自己的专业技术，不断创造。从来都没有想过会在去碰英语的我。对英语这一块不喜欢和无奈的我，为了更好的自己、也考虑到英语的重要性，上网找了一些适合自己也能帮助自己学习的视频（分类归纳记忆法），也找学英语的 App，先把自己的词汇量增加，试着突破自己，看能不能有所改善，不再抗拒而是喜欢..这边也有在提升自己学历，并且也想过后期经济稳定一些，再去自考本科，毕竟大城市还是比较看中学历这块的。

9.9 正式迎来岗位的转变，对于这个岗位来说我是一名小白，但我会尽快让自己上手，不拖部门的后腿。这两周都在学习新东西：注意中英文排版问题；更新应用版本（先检测这个包有没有问题在加固和签名然后在检测有没有问题最后才上传）；测试等。

哦还有一个，我现在也是有属于自己域名：yuuilee.com。刚开始想 id 名字的时候，都有考虑用鱼，鲤鱼的英文单词了，跟我的名字有点像的谐音，还好还好，是朋友阻止了我，觉得取带有这样含义的有点搞笑很幼稚。
